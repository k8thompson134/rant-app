## Enabling GitHub Pages

1. Go to your repo: https://github.com/k8thompson134/rant-app
2. Click **Settings** â†’ **Pages** (in sidebar)
3. Under "Source", select **Deploy from a branch**
4. Under "Branch", select **main** and **/docs** folder
5. Click **Save**
6. Wait 1-2 minutes for deployment
7. Visit https://k8thompson134.github.io/rant-app/ to verify

## Updating Documents

To update these documents:
1. Edit the markdown files in this directory
2. Commit and push to GitHub
3. GitHub Pages will auto-update within 1-2 minutes
